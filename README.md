# Tic-Tac-Toe

Classic tic-tac-toe game built with vanilla JavaScript.

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Game Rules

- Two players: circle (blue) and cross (red)
- Circle goes first
- The first player to align three symbols in a row (horizontally, vertically, or diagonally) wins
- If all cells are filled with no winner, it's a draw
